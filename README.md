# Local AI Stack

Self-hosted AI stack using **Ollama + Qdrant + Open WebUI** on Docker.  
Focus: private RAG, long‑term semantic memory and offline usage.

## Features
- Ollama as local LLM server.
- Qdrant as vector database for long‑term memory.
- Open WebUI as web interface.
- Docker Compose for easy deployment.

## Goals
- 100% local, no cloud dependencies.
- Reproducible lab for cybersecurity and automation experiments.
